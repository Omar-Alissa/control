# Product Management System

A simple CRUD system for managing products.  Built with vanilla JavaScript as a practice project.

## What does it do?

This is a product management system where you can add, edit, and delete products. The system automatically calculates the total price based on price, taxes, ads, and discount.

## Features

- Add products with price, taxes, ads, and discount
- Automatically calculates total price
- Update products
- Delete individual products or all at once
- Search by title or category
- Save data in localStorage

## Installation

Clone the project and open index.html in your browser:

```bash
git clone https://github.com/Omar-Alissa/control.git
cd control
```

Then open `index.html` in your browser. That's it!

## How to use

1. Fill in product information in the form
2. Click "create" to add the product
3. Use the search field to filter products
4. Click "update" to edit a product
5. Click "delete" to remove a product

## Files

- `index.html` - HTML structure
- `style.css` - CSS for styling
- `logic.js` - All JavaScript functions

## Technologies

- HTML
- CSS
- JavaScript
- localStorage for saving data

## Demo

You can test the site here: https://omar-alissa.github.io/control/

## Note

All data is saved locally in your browser with localStorage, so no database is needed.  This means the data will be lost if you clear your browser's cache.
